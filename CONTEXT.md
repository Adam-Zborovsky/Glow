# Glow — Build Your Page. Share Your World.

> One link. Your entire online presence. Beautiful by default.

---

## 1. What Is Glow?

Glow is a personal page builder. You sign up, pick a template, add your links, content, and media — and in under 5 minutes you have a beautiful page at `glow.page/yourname` that you can share everywhere: Instagram bio, Twitter/X, TikTok, LinkedIn, email signature, resume, business cards.

Think Linktree — but actually beautiful, more powerful, and with a free tier that doesn't feel crippled.

**One-sentence pitch:** "Create a stunning personal page in minutes and share one link everywhere."

**Why it's marketable:**
- Linktree has **50 million users** and a **$1.3B valuation** selling this exact concept
- Every page created IS an ad for Glow (a "Powered by Glow" badge)
- Anyone with a social media account instantly understands the value
- The product markets itself — users share their page link on every platform
- Broad audience: creators, freelancers, small businesses, students, job seekers, musicians, podcasters, literally anyone online

## 4. Core Features

### 4.1 Page Builder (The Heart of Glow)

A block-based, drag-and-drop page editor with live preview.

**Blocks Available:**

| Block Type | Description |
|-----------|-------------|
| **Link Button** | A styled button linking to any URL. Custom icon, label, and description. |
| **Link Card** | A richer link with thumbnail image, title, description, and CTA button. |
| **Bio / About** | A text section with your photo, name, title, and a short bio paragraph. |
| **Social Icons** | A row of social media icons linking to your profiles (Instagram, X, TikTok, YouTube, LinkedIn, GitHub, Dribbble, etc.). |
| **Image** | Single image with optional caption and link. |
| **Image Grid** | 2-4 column photo gallery (portfolio, product photos, art). |
| **Video Embed** | YouTube, Vimeo, or TikTok video embed. |
| **Music Embed** | Spotify, Apple Music, SoundCloud track or playlist embed. |
| **Header / Divider** | Section heading text or visual divider to organize your page. |
| **Text Block** | Rich text paragraph for longer descriptions, announcements, or stories. |
| **Newsletter Signup** | Email capture form that connects to Mailchimp, ConvertKit, or stores in Glow. |
| **Contact Form** | Name + email + message form that forwards to your email. |
| **Testimonial** | Quote card with author name, photo, and title. |
| **Map** | Embedded Google Map for a business location. |
| **Countdown** | Timer counting down to an event, launch, or deadline. |
| **PDF / File** | Downloadable file (resume, menu, press kit). |
| **Custom HTML** | (Pro) Embed any custom code — widgets, forms, scripts. |

**Editor Features:**
- Drag-and-drop block reordering
- Real-time live preview (phone mockup on right side)
- Block settings panel (slides in from right when a block is selected)
- Undo / redo
- Mobile preview toggle
- Publish / unpublish toggle
- Copy page URL with one click

### 4.2 Templates & Theming

**Template Library:**
20+ professionally designed templates organized by use case:

| Category | Template Names | Vibe |
|----------|---------------|------|
| **Creator** | "Spotlight", "Broadcast", "Stage" | Bold, colorful, content-forward |
| **Professional** | "Executive", "Folio", "Slate" | Clean, minimal, business-ready |
| **Artist** | "Gallery", "Canvas", "Neon" | Visual, expressive, unique |
| **Music** | "Vinyl", "Backstage", "Setlist" | Dark, moody, music-player integrated |
| **Fun / Personal** | "Sunset", "Pixel", "Cloudy" | Playful, colorful, personality-driven |
| **Minimal** | "Mono", "Paper", "Zen" | Ultra-clean, whitespace-heavy, elegant |
| **Dark** | "Midnight", "Shadow", "Obsidian" | Dark backgrounds, glowing accents |

**Custom Theming:**
- Color picker for background, text, buttons, accents
- Font selector (20+ Google Fonts curated for readability)
- Background options: solid color, gradient, image, animated gradient, pattern
- Button style: rounded, pill, square, outlined, filled, ghost
- Card style: bordered, elevated, flat, glass-morphism
- Spacing control: compact, comfortable, spacious
- Custom CSS (Pro tier)

### 4.3 Analytics Dashboard

Track who visits your page and what they click.

**Metrics:**
- **Page Views**: Total views, unique visitors, views over time (line chart)
- **Click Tracking**: Clicks per link/block, click-through rate, most popular links
- **Traffic Sources**: Where visitors come from (Instagram, Twitter, direct, Google, etc.)
- **Geography**: Visitor locations by country (map visualization)
- **Device Breakdown**: Mobile vs. desktop vs. tablet (donut chart)
- **Referrers**: Top referring URLs
- **Time Range**: Today, 7 days, 30 days, 90 days, all time

### 4.4 Custom Domains

Connect your own domain to your Glow page.

- **Free**: `glow.page/username`
- **Pro**: Connect `yourname.com` or `links.yourbrand.com`
- Simple DNS setup with guided instructions
- Automatic SSL certificate
- Domain verification status indicator

### 4.5 SEO & Social Sharing

Make your page discoverable and shareable.

- Open Graph (OG) image upload or auto-generation
- Twitter Card support
- Favicon customization
- Google indexing toggle
- Structured data (JSON-LD) for rich search results

| **TanStack Query** | Server state management and caching |
| **Framer Motion** | Animations and transitions |
| **next-themes** | Dark/light mode support |
| **react-colorful** | Color picker component |

| Technology | Purpose |
|-----------|---------|

| **PostgreSQL (Neon)** | Database (serverless Postgres) |

## 10. Key User Flows

### Flow 1: New User → Published Page (< 5 min)

```
1. Land on glow.page → see hero + "Get Started Free"
2. Click CTA → signup page
3. Sign up with Google (one click) → pick username
4. Redirected to editor with a default "Spotlight" template pre-loaded
5. Template has placeholder content: bio, social links, 3 sample link buttons
6. User edits: uploads photo, changes name/bio, swaps link URLs
7. Clicks "Publish" in top toolbar
8. Sees celebration modal: "Your page is live! 🎉" with shareable URL
9. Copies URL, pastes in Instagram bio
10. Done. Under 5 minutes from first click to live page.
```

### Flow 2: Existing User → Check Analytics

```
1. Log in → dashboard shows quick stats
2. Click "Analytics" in nav
3. See 30-day views chart, top links, traffic sources
4. Filters by "Last 7 days" to see recent performance
5. Notices "Portfolio" link has most clicks → decides to feature it higher
6. Clicks "Edit Page" → drags Portfolio block to top position
7. Clicks "Publish" → changes are live instantly
```

### Flow 3: Upgrade to Pro

```
1. User tries to connect custom domain → sees "Pro Feature" badge
2. Clicks "Upgrade" → taken to billing settings
3. Sees Pro features list + pricing ($8/mo or $96/yr)
4. Clicks "Start 14-Day Free Trial"
5. Stripe Checkout → enters payment
6. Redirected back → Pro badge appears, domain field unlocked
7. Enters domain, follows DNS instructions, verifies
```

## 13. Stitch Workflow Guide

### Recommended Order for Generating Screens

1. **Start with Screen 09** (Published Page — Creator template) as your reference design
2. Run `extract_design_context` on it to capture the Glow design DNA in Stitch's format
3. Use that extracted context for ALL subsequent screen generations
4. Generate screens in this order for best workflow:
   - Published pages (09, 10, 13) — they're self-contained and help define the brand
   - Marketing pages (01, 02, 03, 04) — landing page sections
   - Auth (11) — login/signup
   - Dashboard (05, 08) — authenticated views
   - Editor (06, 07) — most complex, do last
   - Settings (12) — straightforward

### Tips for Glow-Specific Prompts

1. **Always include the gradient**: `linear-gradient(135deg, #7C3AED, #EC4899)` is Glow's signature
2. **Published pages vary wildly**: Each template has its own vibe — specify colors per template
3. **The editor is the most complex screen**: Consider generating sub-panels separately (left sidebar, canvas, right panel) then composing
4. **Use "Inter" explicitly**: Stitch defaults to system fonts otherwise
5. **Specify the logo**: `"glow" in Inter 20px/800 with gradient text (linear-gradient #7C3AED to #EC4899)`
